# 杂项工具

## DisplayAny
显示任何输入的字符串表示。

### 输入
- `input`: 任意输入

### 输出
- `text`: 输入的字符串表示

## PrimitiveText
创建一个基本的文本字符串。

### 输入
- `string`: 输入文本（多行，默认为空）

### 输出
- `text`: 创建的文本字符串

## FillMaskedImageArea
使用指定的填充值填充图像中被遮罩覆盖的区域。

### 输入
- `image`: 输入图像
- `mask`: 遮罩
- `fill`: 填充值（浮点数，默认为0，范围0-1，步长0.01）

### 输出
- `IMAGE`: 填充后的图像
