# 图像处理工具

## ColorAdjustment
对图片进行色彩校正。

### 输入
- `image`: 输入图像
- `brightness`: 亮度调整值
- `contrast`: 对比度调整值
- `saturation`: 饱和度调整值
- `hue`: 色相调整值
- `gamma`: 伽马值调整

### 输出
- `adjusted_image`: 调整后的图像
